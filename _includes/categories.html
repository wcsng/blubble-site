We are researchers of WCSNG lab at UCSD, and we are collecting data for building a classification algorithm to sense proximity via RSSI values of BLE advertised beacons. For this, we have built a crowdsourced data-collection platform, and we would be very grateful if you could contribute to this. The data we are collection involves Bluetooth RSSI data, geo-location data and some data about the device chipset/manufacturers. All the data is stored in our secure database hosted on Google Firebase, and any data collected is anonymized with randomized user id strings.
