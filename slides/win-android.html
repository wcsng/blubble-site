Overview of Tasks:
1. Setup Quick Beacon app on your windows
device
2. Install the Android app on your mobile device
3. Perform the experiments as per the
instructions in the video attached towards the
end
Search for Quick Beacon in Microsoft
store. Alternately, follow this link. Install the
highlighted application.
Set up Fixed Device (1)
Open Quick Beacon app. Click on Start
Advertising. After clicking, the button will
turn red as shown below
Set up Fixed Device (2)
Download .apk file (download link,
alternately scan the QR code above)
Tap Install
Android App Installation (1)
Tap Allow from this source if you are
installing via the QR code, otherwise
continue ahead
Android App Installation (2)
Tap Install Anyway and Dont
Send in the subsequent window
Android App Installation (3)
Continue to Open the app.
Android App Installation (4)
Open the app you just installed on your
mobile device
Ensure Bluetooth is turned on
Tap Scan For Beacons
