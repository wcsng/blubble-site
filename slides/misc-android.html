Instructions for
Fixed Device: Wearables
Mobile Device: Android
Overview of Tasks:
1. Install the Android app on your mobile device
2. Perform the experiments as per the
instructions in the video attached towards the
end
Devices like Fitbits, Tiles, Apple pencils
beacon regularly by default so that devices
can discover and use them. These devices
beacon by default and do not require any
explicit set up
Set up Fixed Device
● Download .apk file (download link,
alternately scan the QR code above)
● Tap “Install”
Android App Installation (1)
● Tap “Allow from this source” if you are
installing via the QR code, otherwise
continue ahead
Android App Installation (2)
● Tap “Install Anyway” and “Don’t
Send” in the subsequent window
Android App Installation (3)
● Continue to “Open” the app.
Android App Installation (4)
● Open the app you just installed on your
mobile device
● Ensure Bluetooth is turned on
● Tap “Scan For Beacons”
Set up mobile device (1)
a. Select ‘<Your wearable device>’ (eg. it could
show up as device names like Charge 3
Fitbit, Tile etc) in the drop down menu
b. Tap “Start Collecting RSSI” after making
the device selection
