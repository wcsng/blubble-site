Setup and Experiment
Instructions for
Fixed Device: Windows
Mobile Device: iPhone/iPad
Overview of Tasks:
1. Download and run quick beacon app on your
fixed device
2. Install iOS app on the mobile device
3. Perform the experiments as per the
instructions in the video attached towards the
end
● Search for ‘Quick Beacon’ in Microsoft
store. Alternately, follow this link. Install the
highlighted application.
Set up Fixed Device (1)
● Open ‘Quick Beacon’ app. Click on ‘Start
Advertising’. After clicking, the button will
turn red as shown below
Set up Fixed Device (2)
iOS App Installation (1)
● Before you start make sure your phone is updated to
iOS 13.4.1 Software.
● Download Testflight from App Store.(Download
Link)
● Open this Link in Testflight:
https://testflight.apple.com/join/LWw7Hz8x
Or equivalently scan the QR code on the top
● Tap Install and Open the app
iOS App Installation (2)
Set up Mobile Device (1)
● Install the app as per earlier instructions on
your mobile iOS device.
● Open the app and grant access to the
permissions required
● Turn on your Bluetooth
● Tap “Scan for Beacons”
● The fixed device will show up as “Windows
Device”. Select that and continue
The setup for collecting data is now complete.
Please see the video in the next slide to get
details on how to conduct the experiments
